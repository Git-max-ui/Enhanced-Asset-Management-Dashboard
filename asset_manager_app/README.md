# 资产管理 App（Streamlit）

一个轻量的资产管理应用，支持：
- 多用户隔离：不同用户数据完全独立
- 添加 / 修改 / 删除物品
- 字段：名称、种类（含 Logo）、价格、购入日期
- 自动统计：使用天数、日均价格、在役时长、保值率估算
- 汇总统计：总价格、总日均价格
- 分类：内置常用类型 + 自定义种类与 Logo + 智能推荐
- 列表：分类筛选、标签筛选、价格区间、日期范围、导出 Excel
- 搜索增强：建议提示、最近 10 条搜索历史
- 统计页：4 类高级交互图（趋势/热力图/漏斗/雷达）
- 心愿单页：记录想买物品，支持状态管理
- 首页仪表盘：暗色卡片式布局、预警提醒、最近资产、快速操作

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

## 2. 运行

```bash
streamlit run 🏠首页.py
```

兼容旧入口：

```bash
streamlit run 首页.py
```

## 3. 数据存储

- 使用 SQLite，本地数据库文件：assets.db
- 重启应用后数据会保留

## 4. 页面说明

- 首页：增强仪表盘（指标卡、分类环形图、快速操作、标签云、预警）
- 统计分析：资产趋势线、消费热力图、心愿漏斗、健康度雷达图
- 心愿单：想买清单、优先级、状态更新

## 5. 多用户使用

- 在首页左侧边栏输入用户名并点击“切换用户”
- 应用会按用户隔离分类、资产、心愿单数据
- 同一数据库文件中不同用户互不干扰
